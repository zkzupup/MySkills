# MySkills

A collection of agent skills. Each skill is defined as a structured file that describes a capability, its inputs, and how to use it.

## Structure

```
skills/
  <category>/
    <skill-name>.md   # Skill definition and documentation
```

## Skills

| Skill | Category | Description |
|-------|----------|-------------|
| [Example Skill](skills/general/example-skill.md) | General | A template demonstrating the skill format |

## Adding a New Skill

1. Create a new Markdown file under `skills/<category>/`.
2. Follow the format defined in [skills/general/example-skill.md](skills/general/example-skill.md).
3. Add an entry to the table above.
